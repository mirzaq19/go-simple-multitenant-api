# Go Simple Multitenant API

## How to setup

1. Clone the repository
2. created `.env` file by copying `.env.example` and modify the content
3. Running app using this command
   ```sh
    go run .
   ```

   Or using air, you must install first

   ```
   # install air-verse
   go install github.com/air-verse/air@latest

   # Then run it
   air
   ```
